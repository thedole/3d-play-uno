DOK3D My experiments in 3 dimensions

  • Model viewer
This project is about me experimenting with making a 3d-engine of sorts. I will not necessarily use commonly accepted techniques that people who actually know what they are doing use, partly because I don't have a clue about what I'm doing, and partly because half the point is to see what I can make more or less intuitively. I will probably try to make it more in line with how it should be done when I have tried and failed the "intuitive" way, or otherwise feel ready to learn some more.
